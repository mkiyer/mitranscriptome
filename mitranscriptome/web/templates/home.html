About

MiTranscriptome is currently in beta testing. Please report issues at our github site.

MiTranscriptome is a large-scale ab initio transcriptome meta-assembly from 6,503 RNA-Seq libraries. RNA-Seq data were obtained from The Cancer Genome Atlas (TCGA), the ENCODE Project, the Michigan Center for Translational Pathology, and other published sources. The lncRNAs most differentially expressed in cancers and tissues are reported in this portal.

Methods

Large-scale transcriptome meta-assembly was performed using AssemblyLine, which rigorously filters background noise and artifacts before employing a greedy dynamic programming approach to derive a consensus transcriptome from individual samples.

Expression analysis was performed using a novel non-parametric tool called Sample Set Enrichment Analysis (SSEA), which uses weighted Kolmorgorov-Smirnoff-like tests to detect cancer-specific and lineage-specific patterns. SSEA adapts ideas from Gene Set Enrichment Analysis.

Welcome to the MiTranscriptome lncRNA Portal

MiTranscriptome is ab initio transcriptome reconstruction from 6,503 RNA-seq libraries spanning diverse cancer and tissue types. The MiTranscriptome contains >58,000 lncRNAs, the vast majority of which are currently unannotated. This portal reports the lncRNAs that are expressed in a lineage-specific or cancer-specific manner.
